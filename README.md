# Face Detection Using Haar Cascades OpenCV
This Repository is for Motion Face Detection

## Step 1
##### Install OpenCV
Open Terminal:
Write : 
```
pip install opencv-python

```


##### Install NumPy
Open Terminal:
Write : 
```
pip install numpy

```

##### Install Scipy
Open Terminal:
Write : 
```
pip install scipy

```
